Tema 02
produkt

På tema 2 blev blev vi introduceret HTML og CSS, og fik studiestartsprøven udlevert. Prøven var ikke en test af ens kompetencer inden for udvikling af hjemmesider, men mere en test om engagement. Vi skulle arbeje ud fra et Layoutdiagram og et Wireframe som vi havde fået udlevert.

Process

Studiestartsprøven i og for sig selv var relativ lige til; man fik et wirefram og et lauoutdiagram udlevert, og så sku man ellers bare følge det. Der var et fokus på at lære, samt bruge, både Grid og Flex i opgaven, hvortil det udleverede Designdiagram havde alle de klasser man skulle bruge.

Læring

Læringen på dette tema var relativ bred;
- Intro til HTML
- CSS og links
- designprincipper
- CTA's og generelt design
- CSS Grids
- Media Querier
- CSS Flexboc
- Designprincipper, yderpunkter og analyse
- Billedformater
- Git
